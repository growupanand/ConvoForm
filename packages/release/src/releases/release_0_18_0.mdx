## What's new

We're excited to announce our biggest release yet with major new features to enhance your form creation experience!

## Google Forms Import
![image](https://github.com/user-attachments/assets/cf6f1707-f2c0-47fd-9224-fddb6e00c6d9)


You can now import your existing Google Forms directly into ConvoForm! Simply select your Google Form file using the new Google picker, and ConvoForm will convert it for you.

Supported Google Form question types:
- Text questions
- Date questions
- Multiple choice questions 
- Scale/Rating questions

## New Answer Input Types

### Rating Input
![image](https://github.com/user-attachments/assets/791dada1-db36-4c53-917a-bfa21f80846a)

We've added a new Rating input type that allows respondents to provide feedback on a scale. This is fully integrated with analytics to help you visualize response data.

### Enhanced Date Input
![image](https://github.com/user-attachments/assets/2f71af1a-6016-43f2-b930-0aacce98ffc7)

Date inputs now support:
- Natural language date typing (like "tomorrow", "next Friday")
- Time picker functionality for collecting precise timestamps

### Multi-select for Multiple Choice
![image](https://github.com/user-attachments/assets/c983304f-a0e2-473e-ad1d-efc37158bc0a)

Multiple choice questions now support checkbox-style multiple selection, giving respondents the ability to choose more than one option.

## Text Improvements
![image](https://github.com/user-attachments/assets/800c41a7-b3aa-43d5-99fa-96cde33ef5e5)

- Added Paragraph mode for text inputs, making it easier to collect longer responses

## Performance Improvements
- Reduced bundle size significantly by replacing Sentry with PostHog error tracking
  - "First Load JS shared by all" reduced from 237KB to 112KB
