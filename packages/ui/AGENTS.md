# ui

## Purpose
A shared UI component library based on Shadcn UI, Radix UI, and Tailwind CSS. It provides consistent design primitives for the applications.

## Key Files
| File | Description |
|------|-------------|
| `src/index.tsx` | Exports UI components |
| `src/components/ui/` | Individual UI components (Button, Input, etc.) |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `src/components/ui/` | Atomic UI components |
| `src/hooks/` | UI-related hooks |
| `src/lib/` | UI utilities (cn, etc.) |

## For AI Agents

### Working In This Directory
- Use `pnpm ui:add` to add new Shadcn UI components.
- Style components using Tailwind CSS classes.
- Ensure components are accessible (using Radix UI primitives).

## Dependencies

### External
- `react`
- `tailwindcss`
- `@radix-ui/*`
- `class-variance-authority`
- `lucide-react` (Icons)
